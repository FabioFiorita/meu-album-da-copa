## Cursor Cloud specific instructions

This is a single-package Convex + Vite + React 19 PWA ("Meu álbum da Copa", FIFA World Cup 2026 sticker album). There is no separate database/API service — Convex is the backend.

- Run the full stack (Convex backend + Vite frontend together) with `CONVEX_AGENT_MODE=anonymous npm run dev`. The `CONVEX_AGENT_MODE=anonymous` env var is required in this headless environment: it spins up a local anonymous Convex backend without browser login. Without it, `convex dev` tries interactive OAuth and hangs.
- The dev command runs `convex dev --start "vite --open --host"`, which provisions the local deployment, writes `.env.local` (`VITE_CONVEX_URL`, `CONVEX_DEPLOYMENT`, `VITE_CONVEX_SITE_URL`), and starts Vite. First run downloads the local Convex backend binary (one-time, at runtime — not part of dependency install).
- Ports: Vite `http://localhost:5173`, Convex backend `http://127.0.0.1:3210`, Convex dashboard `http://127.0.0.1:6790`.
- `.env.local` is git-ignored and regenerated by `convex dev`; do not commit it.
- Standard commands (see `package.json`): `npm run lint` (tsc + eslint), `npm test` (vitest + convex-test backend suites, no running deployment needed), `npm run build`.
